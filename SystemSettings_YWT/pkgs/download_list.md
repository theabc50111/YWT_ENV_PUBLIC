# Offline Package Download List

This file lists the repositories that need to be downloaded and placed inside the `./pkgs/` directory for the script to run with the `--can-git-clone false` option.

Each repository must be cloned and placed in a directory with the specified "Local Folder Name".

| Repository URL                                                 | Local Folder Name                  |
| -------------------------------------------------------------- | ---------------------------------- |
| `https://github.com/zsh-users/zsh-completions.git`             | `zsh-completions`                  |
| `https://github.com/zsh-users/zsh-autosuggestions.git`         | `zsh-autosuggestions`              |
| `https://github.com/zsh-users/zsh-syntax-highlighting.git`     | `zsh-syntax-highlighting`          |
| `https://github.com/dracula/zsh-syntax-highlighting.git`       | `zsh-syntax-highlighting-dracula`  |
| `https://github.com/dracula/gnome-terminal.git`                | `gnome-terminal`                   |
| `https://github.com/tmux-plugins/tpm.git`                      | `tpm`                              |
